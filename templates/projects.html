Deep learning (Pytorch Implementation)

Nested U-net Architecture for Medical Image Segmentation

Description:

Image segmentation has gained significant traction in medical image analysis. Among various architectures, nested U-Net stands out with promising results and outcomes. It is a deeply supervised encoder-decoder sub-network effectively reduces the semantic gap between input and output feature maps. Nested U-Net utilizes redesigned skip pathways and deep supervision techniques to enhance and optimize the results. Using different loss functions and optimizers, the learning process becomes more accessible for the network model. This involves preprocessing the data, training with a desired architecture and post-processing to validate the input images against ground truth. Our focus is to improve the accuracy and efficiency of our model compared to traditional U-Net architecture. By tuning different hyper parameters, and using different datasets, the model is trained to generalize its learning capability. Nested U-Net outperforms state-of-the-art models in tasks related to image segmentation and analysis. The resulting output segmented images exhibit enhanced and improved quality. This research aims to advance the field by elevating the accuracy and efficiency of medical image segmentation models.

Neural Network Project

Image Classification

Description:

This project involves building an image classification model to distinguish between images of cats and dogs. The process begins with data collection, including downloading a dataset of cat and dog images. Preprocessing techniques like data augmentation are applied to enhance the dataset's diversity and quality. The model is constructed using a Convolutional Neural Network (CNN) architecture, consisting of convolutional, pooling, and fully connected layers. The model is trained using the prepared dataset and evaluated using validation data. The training and validation accuracies and losses are visualized to assess the model's performance. Finally, the model is tested on a separate test dataset, and its ability to correctly classify images is calculated and compared against a predefined threshold for successful completion of the project.

ATM Connected to Bank

Description:

The "ATM Connected to Machine" project is a C++ implementation of a simple Automated Teller Machine (ATM) system connected to a virtual banking machine. The project aims to simulate the essential functionalities of a real ATM, allowing users to perform common banking tasks.

Key Features:
User Authentication
  • Balance Inquiry
  • Cash Withdrawal
  • Deposit Funds
  • Transaction History:
  • Exit Option
    • Implementation Overview:
    The project is implemented in C++ and employs basic programming concepts such as functions, loops, and conditional statements. It utilizes data structures like arrays or linked lists to store user account information and transaction records. The user interface is text-based, where users interact with the system through a series of text prompts displayed on the screen. The system responds accordingly based on the user's input.
